# АИС «Обращения» — Руководство по запуску

## Требования
- Python 3.10+
- Node.js 18+

## Быстрый старт

### 1. Бэкенд (Django)
```powershell
cd backend
python manage.py runserver
```
Сервер запустится на http://localhost:8000

### 2. Фронтенд (React + Vite)
```powershell
cd frontend
npm run dev
```
Приложение будет доступно на http://localhost:5173

## Страницы
- `/` — Публичная форма подачи обращения (без авторизации)
- `/status` — Проверка статуса по номеру (без авторизации)  
- `/login` — Вход для сотрудников
- `/dashboard` — Дашборд (оператор/администратор)
- `/appeals` — Все обращения с фильтрами (оператор/администратор)
- `/my-appeals` — Мои заявки (исполнитель)
- `http://localhost:8000/admin/` — Django Admin
